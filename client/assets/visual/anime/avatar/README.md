# Canonical Anime Avatar

`canonical_anime_avatar.glb` is an original generated technical asset. It is not
copied or extracted from The Legend of Neverland or another game.

Generate it with:

```powershell
.\scripts\build_anime_avatar.ps1
```

Validate the Godot import with:

```powershell
.\scripts\smoke_anime_avatar_asset.ps1
```

Current contract:

- 23 semantic humanoid bones;
- `idle`, `walk`, `sit`, `phone`, and `talk`;
- Godot-generated mesh LOD enabled;
- phone prop attached at runtime to `RightHand`;
- procedural model remains a technical reference, not final production art.
